# 🧮 Custom JavaScript Calculator

A fully functional calculator built from scratch using **HTML, CSS, and JavaScript**. This project started as a simple idea and gradually evolved into a more advanced logic-based calculator supporting multiple operations.

---

## 🚀 Project Journey

I didn’t build this in one go. I started small:

- First, I used `eval()` just to make things work quickly  
- Then I built a **single-operator calculator** to understand basic logic  
- Finally, I upgraded it into a **multi-operation calculator** using token-based parsing

This journey helped me move from “just making it work” to actually understanding how calculations are processed internally.

---

## ⚙️ Features

- Basic arithmetic operations (+, -, ×, ÷)
- Multi-step expression handling
- Custom logic without relying on `eval()` (final version)
- Clear and delete functionality
- Responsive UI (if applicable)
- Clean and minimal design
